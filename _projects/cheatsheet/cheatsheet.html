#each #map #select #inject

#each

each iterates through each element in a data structure. By adding a block, you can specify what changes are made to each element

scrambling_phone_number = [7, 0, 3, 5, 5, 5, 5, 5, 5, 5]
scrambling_phone_number.each { |num| print rand(1..num)}
# => 434112215

#map

map is identical to each. The only difference is that map forever changes the data structure it is called on.

scrambling_phone_number.map { |num| num / 3 }
# => [2, 0, 1, 1, 1, 1, 1, 1, 1, 1]

#select

select iterates through the elements in a data structure, but returns an array of elements that fit conditions you've specified.

area_code = scrambling_phone_number.select { |num| num != 5 }
# => [7, 0, 3]

#inject

inject returns the sum of or product of all numbers in a data structure. It can add or multiple an extra number (ex: 5)

sum = scrambling_phone_number.inject(0) { |total, num| total += num }
# => 45

sum = scrambling_phone_number.inject(5) { |total, num| total += num }
# => 50
